那些心动的龙族语句

发表于 2019-02-23 | 分类于 小说随笔语录 | 浏览 次
在某个时间段看到某句话，那个瞬间，惊艳了你我

每个人心里都藏有一个死小孩（lonely kid hides in heart）

1.别和屌丝谈勇敢和希望。

2.所谓错过，不是错了，而是过了。

3.有些时间点错过一次，就好比错过了一生。

4.我们做了任何错误的决定，结果只能自己承担。

5.总有些时候让人感觉到自己的弱小，因为无能为力。

6.是啊，无论结局如何，这一刻还是很美的。这就够了。

7.对别人残酷的人，首先得对自己残酷，否则只是懦夫。

8.你需要付出的，只是心底里那点小小的温软，从此坚硬如铁。

9.错了就错了吧，一个做错的英雄，至少比什么都不做的笨蛋好。

10.因为你太卑微了，所以想要幸福你要付出十倍的代价…乃至生命。

11.世间一切幸福，皆月影一现的昙花，唯有孤独与痛，常伴在黄昏深处。

12.誓言是世上最靠不住的东西，只有你对别人还有用的时候，别人才会遵守誓言。

13.最后的最后，他们还是见了面，但有些人已经擦肩而过，有些事已经时过境迁。

14.其实每个人都在表演，人生就像是一场戏，你在戏里扮演的总不会是真实的自己。

15.小丑是那种无论心里是开心还是难过别人都看不出来的人，因为他给自己画上了笑脸。

16.每个人生下来都是娇嫩怕痛的，只有吃过苦的人更能忍耐，不是不痛，而是更能忍耐。

17.一个人可以逃避世间的一切魔鬼，但唯有一个是他永远无法拜托的，那就是懦弱的自己。

18.如果一件事你不相信自己能做到，那你就真的做不到！因为你连希望都丢掉了，你又怎么能做到？

19.在亲人眼里，大义灭亲是个何等残酷的词啊，世间应该有那么一个人，你可以为他背叛一切，甚至于公理和正义。

20.不要做会让自己后悔的事，不要让那些爱你的人难过，因为这个世界上，你爱的人固然很少，爱你的人也绝不多。

21.要追一个距离你那么远的女孩，就该不惜工本，不怕失败。成功了是你赚到了，失败了是理所当然，得之我幸失之我命嘛。

22.你在阳光里垂下头，回忆曾亲吻你脸颊的那个天使，即使她不再出现。有些爱稍纵即逝，却铭心刻骨。它在命运里烧成灰烬，却在记忆里拔节生根。

23.首先，所有女孩都是要追的！你不主动你惦记着人家主动跟你表白？其次，对于女孩最重要的无非是幸福感，这个男孩有用没用不是绝对重要的，而是，你能不能给她幸福感！

24.“我们都是小怪兽，有一天会被正义的奥特曼杀死。”是啊，你是小怪兽，可小怪兽也有小怪兽的好朋友，孤独的小怪兽们害怕得靠在一起，但如果正义的奥特曼要来杀你，我就帮你把正义的奥特曼杀死。

25.在这个世界的某个角落，必有为你而生的人，当你站在悬崖尽头时不要失去希望，要多坚持那么一秒钟，等那个人一骑绝尘如狂风闪电般出现在你面前，你将跨上他的马背，即便他是被所罗门囚禁了一千年的魔鬼。

26.所谓无悔之爱应该是那样一种东西吧……未必要完美无缺，未必要有好结果，但多年之后你在人海中忽然抬起头来，见远处她独立于礁石，你忽然惊悸忽然震动忽然潸然泪下，速度快到来不及恨或者悲伤。 只是爱，不后悔。

27.如果喜欢谁，就满世界去找她，别等她来找你，她可能也在等你……别让她等得对你失望了。如果你喜欢的人要嫁人了，就跟她表白一下，就算为此要把她婚车的车胎打爆也没什么，这是你说出来的最后机会。把这个秘密带进棺材没价值，连陪葬品都算不上。

28.什么样的喜欢是对的，什么样的喜欢是错的？那些没有开花的种子，就该被掩藏在土中吗？甚至没有一个春天愿意让他们发芽。在错误的时间里遇到，就能克制自己不喜欢那个人么？是不是仍然会用尽了力气想去接近，想尽办法掩饰自己，甚至伪装成另外一条鱼。

29.最后留在记忆深处的总是些虚无缥缈的东西，就像你记住一个人往往不是因为她的美，很多年后你连她的样子都忘记了，可偶然在人流如织的街头闻到她惯用的香水味，你在惊悚中下意识地回过头去，却只看见万千过客的背影。你这才想起即便刚才和你擦肩而过的确实是她，即便你跟她面面相对，你也未必能认出她今天的样子。

30.楚子航：每个人的一生里，都会遇见某人，喜欢上他

   有些人在合适的时间相遇，就好像在春天遇到花开

   于是一切都会很好…

   而有些人在错误的时间相遇

   就像是在冬天隔着冰看见浮上来换气的鱼

   鱼换完气沉到水下，再也看不见了

   什么结果都没有

   夏弥：其实你在冰面上看到鱼浮上来

   明年冬天如果你还在那里，那鱼还是会上来换气

   那时候你就可以带一把冰镐

   把冰面砸开把鱼捞上来熬汤
